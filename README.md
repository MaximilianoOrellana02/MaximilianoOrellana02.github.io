# MaximilianoOrellana02.github.io

Trabajo Práctico Bootstrap - Facultad de Ingeniería (UNJu)

Este proyecto consiste en el desarrollo de un sitio web moderno y responsivo para una plataforma de turismo, desarrollado para la carrera de Analista Programador Universitario (APU 2008). El objetivo principal es integrar HTML5, Bootstrap y jQuery para crear una experiencia de usuario interactiva y optimizada.
🚀 Características PrincipalesDiseño 100% Responsivo: El sitio se adapta automáticamente a dispositivos móviles, tablets, desktops y pantallas de gran formato.  Interactividad con jQuery: Implementación de animaciones, filtros dinámicos y validaciones en tiempo real.  Modo Oscuro (Dark Mode): Sistema dinámico para alternar entre temas visuales.Optimización de Recursos: Uso de CSS Sprites para reducir las peticiones HTTP al servidor en iconos de redes sociales y botones.  Módulo Educativo: Sección dedicada a la concienciación sobre ciberseguridad y phishing.  
🛠️ Tecnologías UtilizadasHTML5 & CSS3: Estructura y estilos personalizados con variables CSS.Bootstrap 5: Framework para el diseño responsivo y componentes UI (Modales, Navbars, Cards).  jQuery 3.7.1: Biblioteca para la manipulación del DOM y efectos dinámicos.  Iconografía: Integración de Bootstrap Icons y Font Awesome.  Librerías Externas: AOS (Animate On Scroll) para animaciones al desplazar la página. 
📂 Estructura del Sitio
1. HomeNavbar Responsive: Incluye menús desplegables (dropdown) y selector de tema.  Hero Section: Presentación con video de fondo y texto animado mediante jQuery.  Contadores Animados: Estadísticas dinámicas que se activan al hacer scroll.  Testimonios: Carrusel de opiniones de usuarios.
2. DestinosFiltros Dinámicos: Sistema de búsqueda por categorías (Playa, Montaña, Ciudad) usando los métodos .filter(), .hide() y .show() de jQuery.  Grid Adaptativo: Layout que cambia de 1 a 3 columnas según el tamaño de pantalla.  Tablas de Precios: Formato responsivo (table-responsive) para visualización de tarifas.
3. AgenciasFlip Cards: Tarjetas con efecto de rotación 3D implementadas con CSS y jQuery para revelar información extra.  Sistema de Rating: Calificación por estrellas interactiva desarrollada en jQuery.
4. BlogDiseño de Revista: Distribución de contenidos visualmente atractiva.  Comentarios Simulados: Funcionalidad para dejar mensajes con feedback visual inmediato.
5. ContactoValidación en Tiempo Real: Validación de campos (.on('input')) y sanitización de datos antes del envío.  Feedback de Usuario: Uso de spinners de carga y modales de confirmación tras el envío exitoso.
🛡️ Seguridad: Módulo de PhishingEl proyecto incluye un módulo educativo para alertar sobre técnicas de robo de datos.  Simulación Educativa: Un modal interactivo que presenta ejemplos de fraude comunes.  Identificación de Errores: Interfaz diseñada para que el usuario aprenda a detectar señales de fraude mediante feedback dinámico.  Privacidad: El módulo no captura ni almacena datos reales.
